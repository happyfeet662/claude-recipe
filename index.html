WHAT IS CLAUDE CODE

Claude Codeって、なに？

ひとことで言うと「パソコン仕事を代わりにやってくれる、話せるAI」です。

話しかけるだけ

プログラミングの知識はいりません。「これやって」と日本語でお願いするだけで動きます。

面倒を肩代わり

書類づくり・計算・投稿・ホームページ作成など、時間のかかる作業をまとめてお任せ。

おうちで・スマホで

特別な機材は不要。パソコンやスマホがあれば、すきま時間に少しずつ進められます。

3 STEPS

使うのは、たった3ステップ

むずかしそうに見えて、やることはとてもシンプルです。

準備する

お使いのパソコン（またはスマホ）にClaude Codeを入れます。手順は記事でひとつずつ画像つきで解説します。

フォルダごと渡す

整理はしなくてOK。請求書やレシートをフォルダに放り込んで、そのまま渡すだけ。AIが探して読み取ります。

お願いする

あとは日本語でお願いするだけ。AIが下書きや計算、ファイル作りまで進めてくれます。
